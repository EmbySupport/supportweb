    Platform Support

    To see which platforms are currently supported, please see Emby Linux Platform Support.

    Delivery of Beta Builds

    Native Platform Packages

    The app is currently available in the form of native platform packages:

    • DEB
      Ubuntu, Debian, Linux Mint
    • RPM
      OpenSUSE, Manjaro, Fedora
    • PAC
      Arch Linux, Steam OS

    Easy Installation

    The most easy way to install is by using the new Emby Universal Installer

    New Linux Package Repositories

    The installer is automatically setting up the new Emby Package repository. For more information and manual setup, please see Emby Package Repository.

    Other Package Formats

    We are planning to make the Linux app available in additional package formats during the course of the beta:

    • AppImage
      coming soon
    • Snap
      depending on user demand
    • Flatpack
      depending on user demand
