    Emby Windows: Beta Program

    The Windows and Xbox apps are closely coupled and share the same beta program.

    About this Program

    Specification
    Program Status Paused
    Program Mode Periodic
    Communication Public
    Enrollment Method Registration
    Enrollment Status Closed

    Delivery of Beta Builds

    On Windows

    Three ways to get it:

    1. Use the following store link: ms-windows-store://pdp/?productid=9PP75325481B 
    2. The store web-link: https://www.microsoft.com/store/r/9PP75325481B
    3. Go there and click the link: https://wmc.emby.media/x.html
    Note

    Only registered Beta uses will be able to access the beta app. You need to be logged into the MS Store app (or Website) in order to see it

    On Xbox

    • On the Xbox, go to Apps and open "Edge"
    • In Edge, enter the following in the address bar: wmc.emby.media/x.html
    • Click on the link on that page
    • MS Store opens and shows the app entry
    Note

    Only registered Beta uses will be able to access the beta app. You need to be logged into the Xbox with the Xbox Live account that corresponds to the MS account you have specified when registering for the beta program.

    Beta Communication

    All communication goes through the following forum in the Testing Area:

    https://emby.media/community/index.php?/forum/160-windows-xbox-beta/
