    Server-Installation auf Shield TV

    Es gibt mehrere manuelle Schritte, die abgeschlossen werden müssen, um den Emby Server auf dem Shield TV zu installieren.

    Schritt 1: Ermitteln der IP-Adresse des Shield TV

    • Navigieren Sie auf dem Startbildschirm des Shield TV nach oben rechts zum Einstellungs-Symbol neben der Uhr.

    • Öffnen Sie die Einstellungen.

    • Scrollen Sie nach unten zu "Netzwerk & Internet" und wählen Sie diese Option aus.

    • Es sollte Ihnen zeigen, dass das Shield TV verbunden ist und Ihnen die IP-Adresse geben.

    • Notieren Sie sich diese IP-Adresse, da Sie sie später benötigen werden.

    Schritt 2: Ermitteln des aktuellen APK-Links

    Auf einem PC gehen Sie zu https://emby.media/download.html und Sie werden sehen:

    Klicken Sie auf die Option Shield Android TV und Sie erhalten einen Bildschirm ähnlich:

    Merken Sie sich die Dateinummern im obigen Link, die "4.3.0.26" sind, die wir in einem folgenden Schritt benötigen.

    Schritt 3: Downloader installieren

    • Führen Sie auf dem Shield TV die Google Play Store Apps aus,

    • Suchen Sie nach "Downloader by TROYPOINT".

    • Installieren Sie diese App und führen Sie sie dann aus.

    • Wenn die App um Erlaubnis zur Systemzugriff bittet, erteilen Sie bitte die Erlaubnis.

    Schritt 4: Herunterladen der APK

    Auf dem Startbildschirm im "Downloader by TROYPOINT" sehen Sie ein Eingabefeld für den Download-Standort.

    Geben Sie ein:

    https://github.com/MediaBrowser/Emby.Releases/releases/download/4.3.0.26/com.emby.embyserver_4.3.0.26.apk

    Note

    Wenn die Dateinummer aus Schritt eins von diesem Link abweicht, passen Sie bitte die Zahlen in diesem Link an, um sie mit der aktuellen Datei auf der Website aus Schritt 1 übereinstimmen zu lassen.

    Wenn die App um Erlaubnis zur Systemzugriff bittet, erteilen Sie bitte die Erlaubnis.

    Schritt 5: Installation des Emby Servers

    • Nach Abschluss des Downloads haben Sie die Option, die App zu installieren. Bitte tun Sie dies.

    • Wenn Sie diese Option nicht erhalten oder die App später erneut installieren möchten, können Sie das DATEI-Menü auf der linken Seite verwenden, um die APK-Datei schnell zu finden und zu starten/installieren.

    • Nach dem Starten der APK-Datei erhalten Sie einen Bildschirm mit der Frage:

      "Möchten Sie diese Anwendung installieren? Sie erfordert keinen speziellen Zugriff."

    • Bitte wählen Sie INSTALLIEREN, um fortzufahren.

    Nach Abschluss der Installation haben Sie die Möglichkeit, FERTIG oder ÖFFNEN zu wählen. Bitte wählen Sie ÖFFNEN:

    Das Shield TV wird Sie nun auffordern, ihm die Erlaubnis zum Systemzugriff zu erteilen. Bitte ERLAUBEN Sie es.

    Die Anwendung wird Ihnen nun eine URL bereitstellen, um auf den Server zuzugreifen. Wir WERDEN diese Adresse NICHT VERWENDEN, also ignorieren Sie sie.

    Schritt 6: Zugriff auf das Shield TV von einem PC aus

    • Von einem PC aus werden Sie nun eine Verbindung zum Emby Server herstellen, der auf dem Shield TV läuft.

    • Geben Sie http:// ein

    • Geben Sie die IP-Adresse aus Schritt 1 ein

    • Fügen Sie einen Doppelpunkt gefolgt von 8096 hinzu

    Die auf Ihrem PC eingegebene URL wird dann ähnlich aussehen:

    http://192.168.0.14:8096

    Schritt 7: Konfiguration und Einrichtung

    Sie sollten nun einen Bildschirm haben, der so aussieht:

    Herzlichen Glückwunsch, Sie haben den Emby Server erfolgreich installiert und können ihn nun wie jeden anderen Emby Server von einem PC aus konfigurieren. Bitte folgen Sie ab hier der normalen Installationsprozedur.

    Schritt 8: Optional - Verwendung von USB-angehängtem Speicher

    Wenn Sie einen USB-angehängten Speicher auf dem Shield TV verwenden, sollten Sie diesen Nvidia-Artikel zum Schreibzugriff lesen.

    https://nvidia.custhelp.com/app/answers/detail/a_id/4437/~/how-to-write-to-usb-shared-storage-on-shield
