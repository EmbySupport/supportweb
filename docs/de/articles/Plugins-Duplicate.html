    Plugins

    Plugins erweitern die Funktionalität von Emby, indem sie zusätzliche Funktionen und Dienste bereitstellen. Um Ihre Plugins zu verwalten, öffnen Sie das Server-Dashboard und navigieren Sie zu Plugins.

    Meine Plugins

    Die Registerkarte "Meine Plugins" zeigt Ihre installierten Plugins an. Durch Klicken auf das Menü-Symbol bei einem Plugin können Sie es konfigurieren (falls zutreffend) oder deinstallieren.

    Wenn ein Plugin keine Konfiguration erfordert, wird es in seinem Befehlsmenü keinen Einstellungsbutton geben.

    Katalog

    Der Katalog ermöglicht es Ihnen, neue Plugins zu durchsuchen und zu installieren. Einige der verfügbaren Kategorien sind:

    • Kanäle - Internetkanäle bieten Zugriff auf Streaming-Audio oder -Video. Für mehr Informationen siehe Kanäle.

    • Content Provider - Ermöglicht die Hinzufügung neuer Arten von Medien zur Bibliothek, wie Bücher und Spiele, oder das Herunterladen von zugehörigem Inhalt für bestehende Medien.

    • Live-TV - Bietet Zugriff auf Live-TV-Kanäle. Für mehr Informationen siehe Live-TV.

    • Metadaten - Fügt neue Metadaten-Downloader oder -Filter hinzu.

    • Benachrichtigungen - Fügt neue Benachrichtigungsziele für die Verwendung mit der Benachrichtigungsfunktion hinzu. Für mehr Informationen siehe Benachrichtigungen.

    • Soziale Integration - Zugriff auf Funktionen sozialer Netzwerke.

    Plugins installieren

    Um ein Plugin zu installieren, klicken Sie auf eines im Plugin-Katalog. Die Plugin-Detailseite enthält dann einen Installationsbutton.

    Klicken Sie auf den Installieren-Button, um die Installation durchzuführen. Nach Abschluss der Installation muss der Server neu gestartet werden.

    Beliebte Plugins

    Der Pluginkatalog bietet eine große Auswahl an Plugins. Hier sind einige der populärsten Optionen:

    • Auto Box Sets - Erstellt automatisch Filmkollektionen basierend auf Ihrer Filmsammlung
    • Cover Art - Verbessern Sie Ihre Medienbilder
    • GameBrowser - Fügt Unterstützung für Spiele in Emby hinzu.
    • Trailers - Streamen Sie Internet-Trailer
    • Trakt - Bietet Integration mit dem Trakt-Dienst
