    Untertitel-Namensgebung

    Externe Untertitel sollten denselben Dateinamen wie der Film/die Serie verwenden, außer den Erweiterungen, wie unten gezeigt.

    /Movies
   /Home Alone (1990)
      Home Alone.mkv
      Home Alone.srt
      Home Alone.spa.srt
      Home Alone.spanish.srt
    Note

    Wenn Sie Leerzeichen im Untertitel einfügen müssen, halten Sie die ALT-Taste gedrückt und geben den ASCII-Code 255 für ein Leerzeichen im Dateinamen ein.

    Standarduntertitel

    Externe Untertitel können als Standard markiert werden, indem man ".default" hinzufügt.

    /Movies
   /Home Alone (1990)
      Home Alone.mkv
      Home Alone.srt
      Home Alone.spa.default.srt

    Erzwungene Untertitel

    Externe Untertitel können als erzwungen markiert werden, indem man ".forced" oder ".foreign" hinzufügt.

    /Movies
   /Home Alone (1990)
      Home Alone.mkv
      Home Alone.srt
      Home Alone.spa.forced.srt
      Home Alone.spa.foreign.srt

## Untertitel für Hörgeschädigte

Untertitel für Hörgeschädigte können mit ".sdh" markiert werden.

    /Movies /Home Alone (1990) Home Alone.mkv Home Alone.srt Home Alone.spa.sdh.srt

    
## Andere Namen
Es kann manchmal nützlich sein, andere "Namen" im Untertitel zu verwenden, um ihn vom Client aus zu identifizieren. Dies kann nützlich für Verzeichnisse oder Kommentare sein.

    /Movies /Home Alone (1990) Home Alone.mkv Home Alone.English(Commentary).srt Home Alone.Spanish(Commentary).srt

## ISO- und Ländercodes gemeinsam verwenden

Manchmal müssen Sie zusätzlich zum ISO-Code einen Ländercode verwenden. Dies kann der Fall sein, wenn ein Ländername mit verschiedenen Dialekten verbunden ist. Zum Beispiel könnte "Chinesisch" in vereinfachtem oder traditionellem Chinesisch geschrieben sein.

Mit dem Einsatz von sowohl ISO- als auch Ländercodes können Sie dies spezifizieren. zh ist der ISO-Code für Chinesisch. Um zwischen vereinfachtem und traditionellem Chinesisch bei der Benennung Ihrer Untertiteldateien zu unterscheiden, folgen Sie folgendem Format:

|       |                                     |
|-------|-------------------------------------|
| zh-CN | Erweiterung für Festlandchina (vereinfacht) |
| zh-SG | Erweiterung für Singapur (vereinfacht)      |
| zh-TW | Erweiterung für Taiwan (traditionell)        |
| zh-HK | Erweiterung für Hongkong (traditionell)      |
