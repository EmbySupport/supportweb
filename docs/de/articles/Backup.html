    Konfigurations-Backup

    Diese Anleitung zeigt, wie Sie Ihre Einstellungen von einem Emby Server auf einen anderen Computer migrieren. Es gibt zwei Methoden zum Backup von Emby Server-Einstellungen, die wir hier besprechen werden.

    Verwendung des Backup & Restore Plugins

    Wir empfehlen die Verwendung des Backup & Restore Plugins, das diesen Prozess durch automatisierte Arbeit besonders einfach macht. Dieses Plugin erfordert Emby Premiere und kann in unserem Plugin-Katalog gefunden werden.

    Das Backup & Restore Plugin kann Ihnen beim Sichern und Wiederherstellen der folgenden Dinge helfen:

    • Serverkonfiguration
    • Benutzer
    • Benutzerdaten (Wiedergabestand, Favoriten, etc.)
    • Installierte Plugins
    • Plugineinstellungen
    • Wiedergabelisten

    Dies wird keine Bibliotheksinhalte und Metadaten sichern. Um eine permanente Kopie der Metadaten zu behalten, empfehlen wir das Speichern der lokalen Metadaten in Medienordnern zu aktivieren. Das Plugin sichert zurzeit auch nicht die Einstellungen für Live-TV-Aufnahmen oder Serien.

    Verwendung des Backup & Restore Plugins

    • Stellen Sie sicher, dass Ihr Emby Premiere Schlüssel korrekt installiert und validiert ist. Backup & Restore erfordert Emby Premiere.
    • Installieren Sie das Plugin auf Ihrem aktuellen Emby Server.
    • Konfigurieren Sie das Backup-Plugin, indem Sie einen Ordner festlegen, in dem die Backups gespeichert werden. Standardmäßig wird dieser Vorgang alle 12 Stunden ausgeführt und es werden maximal 3 Backups aufbewahrt. Sie können Ihren Ordner überwachen, um sicherzustellen, dass Backups erstellt werden.
    • Installieren Sie das Plugin auf Ihrer neuen Emby Server-Installation, und konfigurieren Sie den Backup-Ordner auf den gleichen Hauptordner, der Ihre Backups enthält.
    • Klicken Sie auf ein Backup, dann gelangen Sie zum Wiederherstellungsbildschirm, wo Sie eine Wiederherstellung ausführen können.

    Hinweis: Falls frühere Backups nicht korrekt angezeigt werden, überprüfen Sie, ob Sie den Hauptordner und nicht den speziellen Ordner ausgewählt haben, der die Backups enthält.

    Wiederherstellung aus einem Backup

    • Gehen Sie zu Plugins
    • Klicken Sie auf das Backup & Restore Plugin
    • Dies ist der Konfigurationsbildschirm, der Ihnen auch die aktuellen Backups zeigt
    • Scrollen Sie nach unten zu "Aktuelles Backup"
    • Klicken Sie auf das Backup, das Sie wiederherstellen möchten.
    • Sie erhalten einen Bildschirm, der Ihnen Optionen für wiederherstellbare Inhalte bietet.
    • Sie können optional nur bestimmte Teile des Systems oder nur für bestimmte Benutzer wiederherstellen. Konfigurieren Sie nach Bedarf.
    • Klicken Sie auf die Schaltfläche "Jetzt wiederherstellen".
    • Fertig!

    Manuelles Backup

    Die folgenden Anweisungen beschreiben, wie Sie eine Emby Server-Installation manuell sichern oder migrieren können.

    Lokalisiere den Programmdaten-Ordner von Emby

    Der Pfad zum Emby Server-Datenordner ist im Emby Server Dashboard zu finden, indem Sie auf das Drei-Punkte-Menü neben dem Servernamen klicken.

    Klicken Sie auf die Option "Serverinformationen anzeigen".

    Der oberste Eintrag zeigt den Hauptspeicherort des Emby Servers.

    Von nun an wird in dieser Anleitung dieser Pfad als /ProgramData bezeichnet. Sichern Sie einfach alles in und unter /ProgramData mit jedem beliebigen Tool oder kopieren Sie alle Dateien an einen Backup-Speicherort.

    Migration zu einem neuen Server

    Wenn Sie zu einem anderen Betriebssystem migrieren, sollten Sie darauf achten, die Verzeichnisse während der Wiederherstellung ordnungsgemäß abzugleichen, da die Verzeichnisnamen und -standorte unterschiedlich sein können.

    • Erstellen Sie ein gutes Backup wie oben beschrieben von Ihrem aktuellen System (alt).
    • Installieren Sie die gleiche Versionsnummer von Emby Server auf dem Computer, der Ihr (neues) System werden soll.
    • Führen Sie den Setup-Assistenten durch und beenden Sie dann Emby Server.
    • Wenn der Speicherplatz ausreicht, machen Sie eine Kopie der Installation an einem anderen Ort auf der Festplatte. Dies ist nur nötig, wenn Sie Zugriffsrechte überprüfen/beheben müssen.
    • Stellen Sie das Backup von Ihrem alten Computer direkt über die aktuelle Installation auf dem neuen System wieder her.
    • Vergleichen Sie die Zugriffsrechte jedes Ordners auf dem neuen System mit der frischen Installationskopie.
    • Passen Sie die Zugriffsrechte bei Bedarf an, damit sie übereinstimmen.
    • Starten Sie Emby Server.
    • Nehmen Sie alle Änderungen an Dateiorten vor, die möglicherweise erforderlich sind.
    • Führen Sie einen vollständigen Bibliotheksscan für alle Bibliotheken durch.
    • Überprüfen Sie die Protokolldateien auf generierte Fehler.

    Nach dem Scan

    Wenn Ihre Bibliothek mit identischen Pfaden wie in der alten Konfiguration eingerichtet wurde, werden im Allgemeinen auch Benutzerdaten sowie Benutzerzugriffsrechte auf die Bibliothek beibehalten. Sie möchten möglicherweise trotzdem den Bibliothekszugriff für jeden Benutzer überprüfen, um sicherzustellen, dass ihr Kanal und Ordnerzugriff wie gewünscht eingeschränkt sind.
