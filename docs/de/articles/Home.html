    Emby Dokumentation Start

    Ihr Ausgangspunkt für die gesamte Emby Dokumentation

    Emby hat zwei verschiedene Teile:

    • Server - Speichern Sie Ihre Mediathek an einem zentralen Ort
    • Client - Clients oder Apps, die Medien vom Emby Server abspielen

    Ein voll funktionsfähiges Emby-System erfordert sowohl einen Server als auch mindestens einen Client-Player. Der Server beinhaltet die Emby-Web-App, die zur Verwaltung des Servers und Ihrer Mediathek genutzt wird. Sie verfügt auch über einen Mediaplayer, sodass die minimale Installation nur des Servers ausreicht, um zu beginnen. Die meisten werden jedoch Emby-Apps auf anderen Geräten installieren wollen, um die Funktionen der Clients zu erweitern.

    Hat Sie jemand zu seinem Server eingeladen?

    Wenn Sie eine Einladung erhalten haben, einen bestehenden Emby-Server zu teilen, können Sie den Server-Installationsprozess überspringen und direkt zur App-Installation übergehen. Der Server wird nur von Personen benötigt, die ihre eigene Mediensammlung hosten möchten. Wählen Sie eine der vielen Apps, die wir anbieten:

    http://emby.media/download/

    Oder probieren Sie die Online-Web-App aus, indem Sie die Anmeldedaten verwenden, die Ihr Emby-Serveradministrator bereitgestellt hat, unter

    http://app.emby.media
