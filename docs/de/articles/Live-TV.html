    Live-TV-Einrichtung

    Emby bietet eine flexible und sehr leistungsstarke Lösung für Live-TV und DVR. Mit Emby’s TV können Sie Live- oder aufgezeichnete TV-Inhalte auf jedes Gerät streamen, das Sie mit einem Emby-Client oder Webbrowser besitzen.

    Sie können Emby nutzen, um die Notwendigkeit von Kabel-Set-Top-Boxen (STBs) zu eliminieren oder Ihren Kabelanbieter vollständig abzuschaffen, indem Sie den Anbieter wechseln und auf OTA- (Over the Air) Übertragungen für lokale Kanäle umsteigen.

    Emby stellt kostenlose Programmdaten (für USA, UK & Kanada) direkt ohne zusätzliche Kosten zur Verfügung.

    Überblick über die Einrichtung

    Die Einrichtung von Live-TV kann in einige Schritte unterteilt werden:

    • Konfigurieren Sie Ihre TV-Tuner oder M3U-basierte IPTV-Quellen
    • Fügen Sie Quellen für TV-Programmdaten hinzu
    • Stimmen Sie Ihr Kanalangebot mit den Programmdaten ab (Kanalzuordnung)

    Konfigurieren Sie Ihren TV-Tuner

    Out of the box unterstützt Emby Server derzeit die folgenden TV-Tuner:

    • HDHomerun Netzwerk-Tuner (Modelle verfügbar für sowohl OTA als auch Kabel. Muss DLNA-Modell 2015+ unterstützen)
    • Hauppauge TV-Tuner (auf Emby Server für Windows - erfordert Win TV 8.5 oder höher)
    • M3U Tuners
      M3U-Dateien (oder URLs). Siehe Beispiele für m3u-Dateien unter http://xmtvplayer.com/build-m3u-file

    Unterstützung für zusätzliche Tuner kann durch die Installation eines Live-TV-Plugins hinzugefügt werden.

    In den meisten Fällen wird Emby Server Ihren HDHomerun automatisch in Ihrem Netzwerk ohne erforderliche Konfiguration erkennen. Sie können auch manuell einen Tuner hinzufügen. Öffnen Sie einfach das Server-Dashboard, navigieren Sie zu Live TV und klicken Sie auf Hinzufügen unterhalb von Tuner-Geräten.

    Fügen Sie eine Quelle für TV-Programmdaten hinzu

    Out of the box unterstützt Emby Server derzeit die folgenden Quellen für TV-Programmdaten:

    • Emby Guide Data
      (Vereinigte Staaten, Kanada und Vereinigtes Königreich)
    • Schedules Direct
      (Nicht für neue Einstellungen verwenden. Funktioniert ab 1. Juni 2021 nicht mehr)
    • Xml TV

    Unterstützung für zusätzliche Quellen kann durch die Installation eines Live-TV-Plugins hinzugefügt werden.

    Stimmen Sie Ihr Kanalangebot mit den Programmdaten ab

    Der letzte Schritt der Live-TV-Einrichtung besteht darin, die Sender Ihres Tuners mit den von Ihnen eingerichteten Programmdaten abzugleichen.

    • Kanalzuordnung für Live-TV

    DVR-Einstellungen (optional)

    Sie können optional einige Optionen ändern, die für DVR-Zwecke verwendet werden. Diese Optionen umfassen:

    • Anzahl der Tage, für die Programmdaten geladen werden

    • Standardaufzeichnungspfade

    • Standard-Vor- und Nachlaufzeiten für Aufzeichnungen. Damit können Sie die Aufzeichnung X Minuten früher starten und Y Minuten später beenden lassen.

    • DVR-Einstellungen

    Emby Live-TV & DVR ist einsatzbereit.

    Herzlichen Glückwunsch, Ihr Emby Server sollte jetzt korrekt konfiguriert und bereit zur Nutzung für Live-TV und DVR sein.
