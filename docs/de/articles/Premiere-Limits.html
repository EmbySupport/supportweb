    Premiere-Grenzen

    Gibt es eine Begrenzung für Emby Premiere?

    Alle Standardmitgliedschaftsoptionen von Emby Premiere sind für den Gebrauch innerhalb eines Haushalts vorgesehen und beinhalten eine Begrenzung auf 30 Geräte, die für Emby Premiere-Funktionen unterstützt werden.

    Aus unserer langjährigen Erfahrung wissen wir, dass diese Standardlizenz für über 95 % unserer Benutzer ausreichend ist. Wenn Sie jedoch mehr Geräte unterstützen müssen, haben wir mehrere erweiterte Geräteoptionen zur Auswahl.

    Das Emby-Konfigurationsdashboard warnt Sie, wenn Sie sich der Gerätegrenze nähern oder diese überschreiten. Daher ist es am besten, einfach zu warten, bis dies angezeigt wird, und dann bei Bedarf eine größere Option zu wählen.

    Wichtige Punkte zum Merken:

    • Sehr wenige Benutzer benötigen tatsächlich mehr Gerätesupport als die Standardlizenz bietet.
    • Die Begrenzung gilt nur für Emby Premiere-Features, sodass nur Apps, die diese Features nutzen, auf das Limit angerechnet werden.
    • Es handelt sich um eine Gerätegrenze, NICHT um eine Grenze für gleichzeitige Verbindungen.
    • Jedes Ihrer Geräte benötigt Internetzugang und speziell Zugang zu mb3admin.com, um die Begrenzung zu validieren.
    • Die Begrenzung ist an einen Emby Premiere-Schlüssel gebunden, nicht an einen Server. Wenn Ihr Schlüssel auf mehr als einem Server verwendet wird, gilt das Limit kumulativ für alle.

    Wie kann ich feststellen, ob das Limit für mich in Ordnung ist?

    Sehen Sie sich die Funktionsmatrix an und notieren Sie die Funktionen, die Sie verwenden und die NICHT in der "Kostenlos"-Spalte stehen. Zählen Sie die Anzahl der Geräte, die Ihre Benutzer nur für diese Funktionen verwenden werden. Selbst wenn diese Zahl nahe an der Grenze von 25 liegt, empfehlen wir, die normale Lizenz für einen Monat des typischen Gebrauchs durch Ihre Benutzer zu testen und zu sehen, ob sie für Sie funktioniert (was sie für fast jeden tut). Ihr Server zeigt an, ob Sie sich der Grenze nähern oder darüber liegen.

    Wie lange zählt ein Gerät zu meinem Limit?

    Das System verfolgt die Gerätenutzung etwa eine Woche lang. Daher müssen Sie nur die Geräte berücksichtigen, die regelmäßig von Ihren Benutzern verwendet werden, wenn Sie das Limit schätzen. Es ist wichtig zu verstehen, dass dies eine Gerätezugriffsgrenze ist, NICHT eine Grenze für gleichzeitige Verbindungen.

    Wie entferne ich ein Gerät von der Liste?

    Das können Sie nicht. Dieses wird automatisch vom System verfolgt. Beachten Sie, dass die Geräte, die im Abschnitt "Geräte" Ihres Dashboards angezeigt werden, nicht Ihre Premiere-Geräte sind. Premiere-Geräte werden nach Schlüssel verfolgt, nicht nach Server, und nur, wenn sie Premiere-Funktionen nutzen.

    Warum kann ich nicht sehen, welche Geräte in meinem Limit gezählt werden?

    Ihr Emby-Server hat eine "Geräte"-Registerkarte, die alle Geräte zeigt, die jemals mit diesem Server verbunden waren. Es ist wichtig zu beachten, dass es sich dabei NICHT um Premiere-Geräte handelt, die zu diesem Limit gehören. Die Gesamtzahl der Premiere-Geräte ist wahrscheinlich viel geringer als die in dieser Registerkarte aufgeführten. Derzeit können Sie nicht genau sehen, welche Geräte zu Ihrem Limit zählen. Der Hauptgrund, warum wir dies derzeit nicht anbieten, liegt in der Sicherheit und Privatsphäre. Damit Ihr Server diese genauen Informationen sehen könnte, müssten wir ein öffentliches API bereitstellen, das diese Informationen liefert. Da wir nicht verlangen, dass Sie ein Konto bei uns haben, um Premiere zu nutzen, gibt es keine Möglichkeit für uns, dieses API auf eine Weise zu sichern, mit der wir uns wohl genug fühlen, es freizugeben. Wir arbeiten daran, Möglichkeiten zu finden, wie wir dies in Zukunft ermöglichen können.

    Zählt mein Server (oder Server) als ein Gerät?

    Nein. Nur Apps, die eine exklusiv für Premiere verfügbare Funktion nutzen, zählen.

    Kann ich Lizenzen stapeln, um mehr Geräte zu bekommen?

    Nein. Die Lizenzen gelten nur für die Anzahl der angegebenen Geräte. Das Upgrade auf 45 Geräte ist genau für 45 Geräte gedacht, nicht für 45 zusätzliche Geräte.

    Kann ich mehr als einen Premiere-Schlüssel kaufen?

    Ja, aber Sie müssen eine andere E-Mail-Adresse für jeden Schlüssel verwenden. Da wir nicht verlangen, dass Sie ein Konto haben, ist die E-Mail-Adresse alles, was wir haben, um Sie und Ihren Schlüssel zu identifizieren. Daher können Sie nicht zwei Schlüssel auf derselben E-Mail-Adresse haben.
