    Emby Connect

    Emby Connect ist ein kostenloser, optionaler Dienst, der es ermöglicht, ein einziges Benutzername/Passwort-Paar zu verwenden, um auf Emby's Foren und Community sowie auf jeden Emby Server zuzugreifen, der für die Anmeldung mit Emby Connect eingerichtet ist.

    Bedeutet dies, dass mein Streaming über die Cloud läuft?

    Nein, alle Verbindungen erfolgen direkt von Ihren Geräten zu Ihrem Emby Server. Der einzige Zweck der Emby Connect-Funktion ist es, Ihren Geräten zu helfen, Ihren Emby Server zu finden, sodass Sie ihn nicht selbst einrichten müssen.

    Benutzerhandbuch

    Beginnen Sie, indem Sie sich bei Emby Connect registrieren. Sobald Sie Ihr Emby Connect-Konto haben, kann es jedem Emby-Administrator übergeben werden, der es zu Ihrem Konto in seinem System hinzufügen wird.

    Klicken Sie auf die ANMELDEN-Schaltfläche, um fortzufahren.

    Füllen Sie diese Seite mit korrekten Informationen aus und klicken Sie auf die SCHÖPFE MEIN KONTO-Schaltfläche. Befolgen Sie die Anweisungen, die Sie per E-Mail erhalten können.

    Ihr Emby Connect-Passwort ändern

    Ihr Emby Connect-Passwort können Sie hier ändern: https://emby.media/community/

    Klicken Sie oben rechts auf Ihren Benutzernamen und dann auf Mein Profil.

    Klicken Sie anschließend auf Profil bearbeiten.

    Ihnen wird dann ein Formular angezeigt, das Sie mit Ihrem aktuellen Passwort und Ihrem neuen Passwort (zweimal) ausfüllen müssen.

    Wenn Sie sich nicht in der Emby Community anmelden können, versuchen Sie, Ihr Passwort hier zurückzusetzen.

    Passwort zurücksetzen

    Falls Sie Ihr Emby Community-Passwort vergessen haben, können Sie es zurücksetzen.

    Klicken Sie auf der Anmeldeseite auf "Ich habe mein Passwort vergessen".

    Ihnen wird dann ein Formular angezeigt, in dem Sie eine Passwortzurücksetzung anfordern können.

    Geben Sie entweder Ihren Emby Community-Benutzernamen oder die E-Mail-Adresse ein, mit der Sie sich angemeldet haben.

    Klicken Sie auf "Ich bin kein Roboter" und dann auf WEITER.

    Sie erhalten dann eine Bestätigungsnachricht, dass eine E-Mail an die eingegebene Adresse oder die mit dem eingegebenen Benutzernamen verknüpfte Adresse gesendet wurde.

    Sobald die E-Mail eingegangen ist (überprüfen Sie auch Ihren SPAM-Ordner), befolgen Sie einfach die darin enthaltenen Anweisungen.
