    Kamera-Upload

    Der Kamera-Upload ermöglicht das automatische Hochladen von Fotos, die mit mobilen Geräten aufgenommen wurden, auf Ihren Emby Server. Dafür müssen die Emby-Apps auf Ihren mobilen Geräten installiert sein und die Funktion unterstützen.

    Derzeit unterstützen die folgenden Apps das automatische Kamera-Upload:

    • Android
    • Emby für iOS

    Kamera-Upload aktivieren

    Um den Kamera-Upload für eine Emby-App zu aktivieren, müssen Sie ihn zuerst im Einstellungsmenü der App aktivieren.

    Anpassung des Upload-Speicherorts

    Es gibt einige Optionen, die es Ihnen ermöglichen, zu steuern, wohin die hochgeladenen Fotos gespeichert werden. Standardmäßig werden sie in einem internen Datenordner gespeichert, und alle Geräte laden in denselben Ordner hoch. Sie können den Upload-Ordner leicht auf einen benutzerdefinierten Pfad ändern und/oder auch einen Unterordner für jedes Gerät erstellen lassen.

    Um auf diese Einstellungen zuzugreifen, öffnen Sie das Server Dashboard von Emby und navigieren Sie zu Geräte -> Kamera-Upload.

    Hinweis: Bei Verwendung eines benutzerdefinierten Upload-Pfades müssen Sie diesen Pfad zur Server-Bibliothek hinzufügen.

    Es ist auch möglich, einen benutzerdefinierten Upload-Pfad individuell für ein Gerät zuzuweisen. Weitere Informationen dazu finden Sie in den Geräteeinstellungen.

    Hochladen von Apps

    Das Hochladen erfolgt automatisch und erfordert keine Intervention des Benutzers. Bitte beachten Sie, dass einige Apps nur unter bestimmten Bedingungen hochladen, wie z.B. im WLAN, bei einer bestimmten Verbindungsgeschwindigkeit oder bei einem bestimmten Akkuladestand, ebenso wie andere gerätespezifische Faktoren.
