    Shield TV Direkter Datei-Zugriff

    Note

    Wenn man Android 11+ verwendet, ist diese Option jetzt nur noch für externe Player verfügbar.

    Um den direkten Datei-Zugriff auf Android TV für Emby einzurichten, müssen Sie zunächst Freigaben und UNC-Zugriff für Ihre Bibliotheken einrichten, was in "Optionale Netzwerkpfade" behandelt wird.

    Als nächstes müssen Sie die "Android TV Storage Attachment" einrichten, damit Ihr Gerät auf Ihre Mediendateien zugreifen kann.

    Um den direkten Datei-Zugriff auf Ihrem Android TV zu nutzen, müssen Sie sicherstellen, dass Sie Version 1.8.36G oder höher verwenden. Sie finden Ihre Version in Emby, indem Sie auf das Einstellungssymbol oben rechts klicken und dann die Version auf der nächsten Bildschirmseite rechts unten ansehen.

    Von dem oben gezeigten Einstellungsbildschirm klicken Sie auf Wiedergabe

    Aktivieren Sie nun die Option "Pass Direct Path" (sollte aktiviert sein).

    Sobald Sie diese Schritte durchgeführt haben, sollten Sie in der Lage sein, Emby direkt auf Ihre Dateien über "Stats for Nerds" zugreifen zu sehen.
