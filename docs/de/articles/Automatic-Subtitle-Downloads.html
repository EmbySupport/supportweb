    Automatischer Untertitel-Download

    Emby kann Untertiteldateien für Ihre Medien automatisch herunterladen. Es gibt mehrere Untertitel-Plugins, die verwendet werden können.

    Installieren Sie ein Untertitel-Plugin

    Bevor Sie den automatischen Download von Untertiteln einrichten können, müssen Sie ein Plugin wie Open Subtitles einrichten. Bitte stellen Sie sicher, dass Sie Ihr Plugin konfiguriert haben, bevor Sie fortfahren.

    Konfigurieren Sie Ihre Bibliotheken zum Herunterladen von Untertiteln

    Sobald Sie ein Untertitel-Plugin installiert, den Server neu gestartet und das Plugin konfiguriert haben, sind Sie bereit, jede Ihrer Bibliotheken zur Nutzung des Plugins zu konfigurieren.

    Sie können nun eine neue Bibliothek bearbeiten oder einrichten. Wenn Sie eine Bibliothek bearbeiten, stellen Sie sicher, dass Sie die Option "Erweiterte Einstellungen anzeigen" oben rechts anklicken. Sie können jetzt nach unten scrollen, um die folgenden Elemente zu finden.

    Oben sind zwei Optionen dargestellt. Sie sollten die Sprachpräferenz für alle Untertitel einstellen, die Sie herunterladen möchten. Das obige Bild zeigt, dass englische Untertitel zum Herunterladen ausgewählt sind.

    Die zweite Option zeigt alle Untertitel-Plugins an, die Sie installiert haben. In diesem Fall wird nur Open Subtitles angezeigt und ausgewählt.

    Das obige Bild zeigt mehrere Optionen, die Sie konfigurieren können. Diese sollten selbsterklärend sein, mit möglicherweise einer Ausnahme für "Hash-Übereinstimmung erforderlich". Wenn diese Option aktiviert ist, wie gezeigt, werden Untertitel NUR dann heruntergeladen, wenn Ihre Mediendatei EXAKT "Byte für Byte" mit der Datenbank von Open Subtitles übereinstimmt. Wenn Sie Ihre eigenen Titel rippen oder den DVR-Dienst zum Aufzeichnen von Dateien verwenden, erhalten Sie niemals eine Hash-Übereinstimmung und können diese Option nicht verwenden.

    Untertitel werden bei Bedarf heruntergeladen, wenn sie verfügbar sind, über die Geplante Aufgabe "Untertitel herunterladen".

    Wenn Untertitel nicht automatisch heruntergeladen werden

    Wenn Untertitel nicht automatisch heruntergeladen werden, sollten Sie die oben aufgeführten Optionen überprüfen. Beispielsweise erfordert "Hash-Übereinstimmung erforderlich", dass Ihre Mediendatei exakt (Byte für Byte) mit der Quelldatei des Untertitels übereinstimmt. Diese Option kann, wenn sie aktiviert ist, Ihre Chancen auf das Herunterladen von Untertiteln stark verringern, stellt jedoch sicher, dass der Untertitel "perfekt" für Ihre Datei ist.

    Sie sollten auch den folgenden Link überprüfen, um zu sehen, wie Sie Manuelle Untertitel-Downloads verwalten können.

    Tipps

    Emby unterstützt viele Untertitel-Anbieter, darunter: Addic7ed, NapiSub, OpenSubtitles, Podnapisi & SubDb. Die aktuelle Liste finden Sie im Plugins-Katalog.
