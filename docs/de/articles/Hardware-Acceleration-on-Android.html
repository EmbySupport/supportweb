    Hardware-Beschleunigung auf Android

    Emby unterstützt die folgenden Hardware-Beschleunigungsvarianten auf Android:

    • Android MediaCodec
      MediaCodec ist Androids integrierte API für Video- und Audio-Codierung.

    • OpenMax
      OpenMax ist eine API-Spezifikation, die verschiedene Aspekte der Medienbeschleunigung abdeckt. Auf Android ist es die Standard-API für Codec-Implementierungen, während Anwendungen typischerweise MediaCodec verwenden.

    Dies gilt auch für Emby. Emby unterstützt hardwarebeschleunigte Codecs über die MediaCodec API. Wir erwähnen OpenMax hier, um in dieser Angelegenheit Verwirrung zu vermeiden, da Codecs oft Namen wie "h26x.OMX.Google.encoder.*" tragen, wobei OMX die API der Codec-Implementierung angibt, selbst wenn sie über die MediaCodec API genutzt wird.

    Android MediaCodec

    Die Android MediaCodec API deckt eine breite Palette von hardwarebeschleunigten Codec-Implementierungen ab. Bei Android liegt es am Gerätehersteller, geeignete Codec-Implementierungen bereitzustellen (oder nicht).

    Unterstützte Beschleunigungen

    Die folgenden Codecs werden derzeit von Emby auf Android unterstützt. Bitte beachten Sie, dass Emby möglicherweise nicht alle vom Gerät angebotenen Beschleunigungen unterstützt und dass nicht alle Geräte alle Beschleunigungen unterstützen werden.

    • Decoder
      • H.264 (AVC)
      • H.265 (HEVC)
      • MPEG2
      • MPEG4
      • VC1
      • VP8
      • VP9
    • Encoder
      • H.264 (AVC)

    Hardware-Anforderungen

    Die Hardware-Beschleunigung auf Android hängt von der tatsächlichen Hardware und der Unterstützung durch den Gerätehersteller ab. Aufgrund der Vielzahl verfügbarer Geräte ist es kaum möglich, Kompatibilitätslisten zu erstellen und zu pflegen. Im Allgemeinen können wir sagen, dass wir Emby Server erfolgreich mit beschleunigten Codecs von folgenden Herstellern getestet haben:

    • Nvidia (Tegra)
    • Samsung (Exynos)
    • AmLogic
    • RealTek (dies bedeutet nicht, dass es mit jedem Gerät von jedem dieser Hersteller funktioniert)

    Einrichtung und Hardware-Erkennung

    Es ist keine Einrichtung erforderlich. Emby erkennt automatisch die verfügbaren hardwarebeschleunigten Codecs. Diese können angezeigt werden, indem man zur Transcoding-Seite auf dem Server-Dashboard navigiert und im Beschleunigungstyp-Dropdown-Menü "Erweitert" auswählt.
