    Manuelle Untertitel-Downloads

    Emby ermöglicht es Ihnen, Untertiteldateien für Ihre Medien bei Bedarf manuell herunterzuladen. Es gibt mehrere Untertitel-Plugins, die verwendet werden können.

    Installieren eines Untertitel-Plugins

    Bevor Sie das Herunterladen von Untertiteln einrichten können, müssen Sie ein Plugin wie Open Subtitles einrichten. Bitte stellen Sie sicher, dass Sie Ihr Plugin konfiguriert haben, bevor Sie fortfahren.

    Manueller Untertitel-Download

    Klicken Sie bei einer Videodatei auf das Drei-Punkte-/Mehr-Menü und wählen Sie aus dem Popup-Menü die Option "Untertitel bearbeiten" aus.

    Sie erhalten einen neuen Bildschirm mit einigen Optionen. Wählen Sie die Sprache

    und alle benötigten Optionen aus und klicken Sie dann auf die Lupe/Suchsymbol.

    Die Seite wird erweitert und zeigt alle gefundenen Untertitel an, die Ihren Ergebnissen entsprechen, und sieht folgendermaßen aus.

    Alles, was Sie nun tun müssen, ist auf den Download-Pfeil in der Wolke auf der rechten Seite des gewünschten Untertitels zu klicken. Sobald Sie darauf klicken, stellt der Emby Server die Datei in die Download-Warteschlange und sie wird sehr schnell zur Verwendung bereit stehen.

    Note

    Wenn Sie keine Untertitel zum Herunterladen sehen (wie oben), bedeutet das, dass Emby keine Untertitel finden konnte, die Ihren Kriterien entsprechen. Sie können versuchen, einige der Optionen zu ändern oder möglicherweise nach "englischen" Untertiteln zu suchen (falls nicht bereits geschehen). Einige Filme haben möglicherweise keine Untertitel verfügbar.

    Wenn Sie Probleme beim Finden von Untertiteln haben, überprüfen Sie bitte, ob der Film/die Show korrekt identifiziert wurde. Sie können die IMDB-ID manuell mit dem Metadaten-Manager überprüfen. Scrollen Sie dazu nach unten zum Abschnitt "Externe ID" und überprüfen Sie, ob die IMDB-ID korrekt ist. Dieser Abschnitt sieht folgendermaßen aus.
