    Web Client

    Die Emby Web App ist die Hauptschnittstelle zwischen Ihnen und Ihrem Emby Server, mit der Sie Servereinstellungen verwalten, Inhalte mit Freunden teilen und Mediainhalte ansehen können.

    Einrichtungsanleitung

    Die Emby Web App ist in Ihrer Installation des Emby Servers enthalten.

    Um darauf zuzugreifen, öffnen Sie einfach einen Webbrowser und besuchen Sie: http://{Ihre Server-IP}:8086.

    Serververwaltung

    Die Emby Web App ist ein praktischer Ort, um alle Aspekte Ihres Emby Servers zu verwalten, einschließlich der Bibliothekskonfiguration, Benutzer, Plugins und mehr.

    Chromecast und Fernbedienung

    Die Emby Web App unterstützt das Übertragen zu Chromecast-Geräten und anderen Emby-Apps. Um sich mit Ihrem Chromecast-Gerät zu verbinden, klicken Sie einfach auf das Übertragungssymbol in der oberen rechten Ecke der App. Sie werden dann gefragt, mit welchem Gerät Sie sich verbinden möchten. Sobald die Verbindung hergestellt ist, wird jeder Inhalt, den Sie abspielen, an das Chromecast-Gerät gesendet. Sie können einzelne Dateien, ganze Ordner, Shuffle, Instant Mix, Warteschlange und mehr abspielen. Weitere Informationen finden Sie unter Chromecast.

    Direkte Wiedergabe von Medienformaten

    Die Emby Web App kann Medien direkt an Ihren Browser wiedergeben, wenn der Browser das Format nativ unterstützt. Bitte überprüfen Sie Ihren bevorzugten Webbrowser auf unterstützte Medienformate.

    Beste Praktiken für direkte Wiedergabe

    Lassen Sie die Streaming-Bitrate-Einstellung der App nach Möglichkeit auf dem Standardwert Auto. Die App führt Bandbreitentests mit Ihrem Emby Server durch, um die maximal spielbare Bitrate zu bestimmen.

    Wenn Sie die Bitrate-Einstellung anpassen, müssen Sie die Bitrate Ihrer Dateien mit der Bitrate-Einstellung in der App vergleichen. Die Bitrate einer Datei können Sie finden, indem Sie die Medieninformationen in der Weboberfläche prüfen. Wenn die Bitrate einer Datei höher ist als die Einstellung in der App, wird Transcodierung erforderlich sein. Eine Erhöhung der Bitrate-Einstellung in der App kann helfen, die Transcodierung zu reduzieren, kann jedoch die Wiedergabeleistung beeinträchtigen, wenn Ihre Netzwerkverbindung nicht schnell genug ist, um sie zu bewältigen.
