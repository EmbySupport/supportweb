    Benachrichtigungen

    Server-Benachrichtigungen ermöglichen es Ihnen, über die Aktivitäten Ihres Servers informiert zu bleiben.

    Benachrichtigungsdienste

    Benachrichtigungsdienste stellen die Methoden dar, in denen Benachrichtigungen versendet werden. Benachrichtigungsdienste sind über eine Reihe von Plugins verfügbar. Um einen Benachrichtigungsdienst zu installieren, öffnen Sie das Server-Dashboard, navigieren Sie zum Plugin-Katalog und scrollen Sie nach unten zum Benachrichtigungsbereich. Es stehen mehrere Optionen zur Verfügung:

    Die Installation eines Benachrichtigungsdienstes ist genauso wie bei jedem anderen Plugin. Lesen Sie die Anweisungen für das Plugin. In einigen Fällen geht es lediglich darum, innerhalb der Benachrichtigungen des Emby Servers die Benachrichtigungsschnittstelle einzurichten. Für weitere Informationen zur Installation und Konfiguration von Plugins siehe Plugins.

    Konfigurieren von Benachrichtigungen

    Nachdem Sie die Benachrichtigungsdienste ausgewählt haben, die Sie verwenden möchten, ist es Zeit, die einzelnen Benachrichtigungen zu konfigurieren, die Sie erhalten möchten. Beginnen Sie mit dem Öffnen des Server-Dashboards, navigieren Sie zu Benachrichtigungen und wählen Sie + Benachrichtigung hinzufügen.

    Die Seite zeigt die verfügbaren installierten Plugins für Benachrichtigungsdienste an, beispielsweise wenn E-Mail-, Slack- und Webhooks-Plugins installiert sind:

    Um eine Benachrichtigung zu konfigurieren, klicken Sie einfach darauf. Sie werden dann zur Konfigurationsseite der Benachrichtigung weitergeleitet. Es gibt eine Reihe von Möglichkeiten, wie Benachrichtigungen angepasst werden können, nachdem die spezifische Konfiguration des Benachrichtigungsdienstes eingerichtet und getestet wurde.

    Das Folgende bietet die verschiedenen Benachrichtigungsereignisse, die ausgewählt werden können.

    Und Sie können die Ereignisse mithilfe der verfügbaren Dropdowns für Benutzer, Bibliotheken und Geräte einschränken.
