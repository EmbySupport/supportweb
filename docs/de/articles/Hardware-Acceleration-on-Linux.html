    Hardwarebeschleunigung unter Linux

    Emby unterstützt die folgenden Varianten der Hardwarebeschleunigung unter Linux

    • Nvidia NVDEC & NVENC API zur Beschleunigung der Video-Codierung und -Decodierung mit Nvidia GPUs

    • VA API Video Acceleration API für Linux wird von mehreren Geräteherstellern unterstützt

    • Intel QuickSync Video Intels Marke für seine dedizierte Hardware zur Video-Codierung und -Decodierung

    Nvidia NVDEC & NVENC

    Nvidia bietet zwei Hardwarebeschleunigungs-Schnittstellen:

    • NVENC API zur Beschleunigung der Video-Codierung
    • NVDEC API zur Beschleunigung der Video-Entschlüsselung (früher NVCUVID API genannt)

    NVIDIA GPUs enthalten einen oder mehrere hardwarebasierte Decoder und Encoder (separat von den CUDA-Kernen), die voll beschleunigte hardwarebasierte Video-Dekodierung und -Codierung für mehrere beliebte Codecs bieten. Durch das Auslagern der Dekodierung/Codierung sind der Grafikprozessor und die CPU für andere Operationen frei.

    Die Hardwarebeschleunigungs-Engines für die Video-Dekodierung (als NVDEC bezeichnet) und die Video-Codierung (als NVENC bezeichnet) unterstützen eine schnellere als Echtzeit-Videoverarbeitung, die sie nicht nur für die Videowiedergabe, sondern auch für Transkodierungsanwendungen geeignet macht.

    Hardware-Anforderungen

    Die Hardwarebeschleunigung ist für die meisten Nvidia Geräte ab GPUs der Kepler-Generation (z.B. GeForce GT 630) verfügbar, sowohl für Verbraucher- als auch für professionelle Geräte. Detaillierte Informationen zur unterstützten Hardware finden Sie in Nvidias GPU Support Matrix.

    Bemerkungen

    • Nvidia-Treiber, die in Linux-Distributionen enthalten sind oder von diesen bereitgestellt werden, enthalten oft nur einen Teil der Nvidia-Treiber. Es wird immer empfohlen, die vollständigen Linux-Treiber von der Nvidia-Website zu installieren.
    • Emby unterstützt den Betrieb ohne Bildschirm für Nvidia.
      Es ist nicht erforderlich, einen Monitor anzuschließen.

    VA API

    VAAPI (Video Acceleration API) ist eine Open-Source-Bibliothek und API-Spezifikation, die Zugriff auf die Hardwarebeschleunigungsfähigkeiten von Grafikhardware für die Videobearbeitung bietet. Sie besteht aus einer Hauptbibliothek und treiberspezifischen Beschleunigungs-Backends für jeden unterstützten Hardwareanbieter.

    Intel QuickSync Video

    Intel® Quick Sync Video nutzt die dedizierten Medienverarbeitungsfähigkeiten der Intel® Graphics Technology, um schnelle Video-Dekodierung und -Codierung zu ermöglichen. Dies beschleunigt die Verarbeitungsgeschwindigkeit und verbessert die Systemreaktionsfähigkeit, da der Prozessor andere Aufgaben schneller erledigen kann.
