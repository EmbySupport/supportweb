    AutoBoxSets

    Erstellen Sie automatisch Sammlungen für Ihre Filme. Sie müssen keine speziellen Ordner mehr erstellen oder Ihre eigenen Sammlungen verwalten. Dieses Plugin hält alle Ihre Filme basierend auf den gleichen TmdbCollectionId-Daten von The Movie DB für Sie in Sammlungen organisiert.

    Um das Plugin zu installieren, klicken Sie im Emby Server Dashboard im Seitenmenü auf Plugins und wählen Sie die Registerkarte Katalog.

    Scrollen Sie nach unten zum Abschnitt Allgemein der Seite und klicken Sie auf das Plugin-Bild.

    Klicken Sie auf Installieren.

    Nach der Installation informiert Sie Emby, dass der Server neu gestartet werden muss.

    Wenn der Server neu gestartet wurde, klicken Sie im Seitenmenü des Emby Server Dashboards auf Plugins und wählen Sie die Registerkarte Meine Plugins.

    Hier sehen Sie das AutoBoxSets Plugin aufgelistet (zusammen mit allen anderen bereits installierten Plugins), wo Sie die Einstellungsseite öffnen können, indem Sie auf das AutoBoxSets-Kunstwerk klicken, um das Menü aufzurufen.

    Sie können eine Aktualisierung der AutoBoxSets manuell mit dem Button Box Sets aktualisieren erzwingen.

    Fehlerbehebung

    Falls Filme nicht wie erwartet zur Sammlung hinzugefügt werden, überprüfen Sie, ob die TmdbCollectionId-ID für Ihre Medien auf der The Movie DB Website korrekt ist. Überprüfen Sie auch, ob Ihre Medien dieselbe TmdbCollectionId in ihren eigenen Metadaten eingestellt haben. Wenn nicht, versuchen Sie, die Metadaten der einzelnen Medien, die Sie der Sammlung hinzufügen möchten, manuell zu aktualisieren.

    Nachdem dies bestätigt wurde, können Sie eine Aktualisierung der AutoBoxSets über die Schaltfläche Box Sets aktualisieren auf der Konfigurationsseite des AutoBoxSets-Plugins manuell erzwingen (siehe oben).

    Hinweis: Wenn Sie nach einem Upgrade auf Emby Server Version 4 Probleme mit AutoBoxSets haben, stellen Sie sicher, dass Sie einen vollständigen Scan durchführen, indem Sie auf die Schaltfläche "Alle Bibliotheken scannen" oben in Ihrer Bibliotheksliste im Menü Bibliothek klicken.
